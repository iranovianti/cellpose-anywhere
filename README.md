# Cellpose Anywhere

A portable Gradio-based GUI for running Cellpose segmentation.

## Quick Start

```bash
pip install -r requirements.txt
python app.py
```

## Usage

1. Upload images
2. Select channel(s) for segmentation
3. Click "Run Cellpose"
4. Download ROIs for ImageJ

See `requirements.txt`
